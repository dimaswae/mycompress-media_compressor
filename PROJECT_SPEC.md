# PROJECT_SPEC.md

## Project Scope

Build a web application that demonstrates multimedia compression and steganography techniques.

## Functional Requirements

### Image Module

Compression:

* RLE
* Huffman Coding

Steganography:

* LSB

### Audio Module

Compression:

* Bitrate Reduction

Steganography:

* LSB on WAV samples

### Video Module

Compression:

* FFmpeg transcoding

Steganography:

* Frame-based message embedding

### Security

Optional:

* AES Encryption before embedding

## Non Functional Requirements

* Responsive UI
* Maximum upload size 100 MB
* Processing time less than 30 seconds for normal files

## Deliverables

* Working application
* Source code
* Technical report
* Presentation video
